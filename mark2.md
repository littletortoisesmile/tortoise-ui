
### 1.数组格式

```js
const nodes = [
    {id: 1, label: 'Node 1', color: 'red'},
    {id: 2, label: 'Node 2', color: 'green'},
    {id: 3, label: 'Node 3', color: 'blue'},
]
const edges = [
    {id: 1, from: 1, to: 2},
    {id: 2, from: 2, to: 3},
    {id: 3, from: 3, to: 1},
]
const datas = {
    nodes: nodes,
    edges: edges,
}
```
